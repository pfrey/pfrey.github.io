PIXEL/code Brand Guide

1. Logo Usage

Always use the full PIXEL/code logo with the slogan when possible. Maintain clear space equal to the height of the "P" around the logo. Avoid stretching, recoloring, or rotating the logo.

2. Color Palette

3. Typography

PIXEL: Retro Pixelated Font (like "Press Start 2P" or custom pixel fonts).

/code: Monospace font (like "Fira Code" or "Courier New").

Backup Fonts: System monospace fonts if custom fonts are unavailable.

4. Brand Voice & Messaging

  • Confident yet friendly
  • Creative + tech-savvy
  • Key Messages: "Custom-built, never cookie-cutter", "Blending creativity with technology"

5. Design Elements

  • Use modular grids
  • Incorporate pixel accents sparingly
  • Soft blue accent for highlights
  • Minimalistic, clean imagery
